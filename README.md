#### What is YelpPal?
- YelpPal is a reviewing website for MealPal restaurants. Where you can find the best rated and best portion meals from others without wasted meals.

-------------------------------------------

#### Technologies Stack
- Node.JS
- Express
- Mongoose/MongoDB
- Bcrypt/Express-Session
- Bootstrap
- External API

#### Code style
- AirBnb JS

-----------------------------------------------------------------------------------
#### User Story
After signing up or in, the user is able to view and add reviews of restaurants connected with MealPal. When the user adds a review, they can add what they got and rate how it was.

-----------------------------------------------------------------------------------
#### Existing Features
- Add Review
- Update Review
- Delete Review
- Login with authentification
